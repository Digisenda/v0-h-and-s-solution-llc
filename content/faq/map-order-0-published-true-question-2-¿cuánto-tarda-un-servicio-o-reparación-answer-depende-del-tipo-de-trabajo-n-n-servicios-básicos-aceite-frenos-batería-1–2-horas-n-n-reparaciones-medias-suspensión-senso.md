question: 2. ¿Cuánto tarda un servicio o reparación?
answer: >-
  \* Depende del tipo de trabajo:


  \* Servicios básicos (aceite, frenos, batería): 1–2 horas.


  \* Reparaciones medias (suspensión, sensores, alternador): mismo día.


  \* Reparaciones mayores (motor, transmisión): se entrega un estimado antes de comenzar.


  \* Siempre priorizamos rapidez sin comprometer la calidad.
